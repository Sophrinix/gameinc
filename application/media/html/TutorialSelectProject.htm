Для создания игры нужен игровой движок, если он есть то можно перейти к процессу создания проекта игры
Игровой движок включает в себя разные технологии с низким уровнем вхождения, которые доступны разработчикам
с самого начала проектирования игры. Тогда как технологии, которые не поддерживаются в движке обходятся разработчикам
значительно тяжелее, сказывается необходимость их адаптации к возможностям движка. Начать процесс создания игрового движка
можно здесь
