Хит-парад или чарт игр — обновляемый каждый месяц список наиболее популярных на данный момент игры. Список хит-парада состоят не более, чем из 10-20 пунктов, располагающихся по мере убывания показателей ( количество продаж, прибыль ). Параметры хит-парада определяются информацией о продажах. Двойной клик на выбранной строке хит-парада откроет окно журнала с описанием игры и рецензиями журналов.
